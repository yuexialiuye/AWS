# 第二周
## SNS
* 建立在發行者跟訂閱者的基礎上
* 通訊分成同步與非同步
## SNS架構
* Queue對應著不同主題
* 發佈者:根據內容不同，把東西放在相對應的Queue裡。
* 訂閱者:根據訂閱的主題收到不一樣的消息。
