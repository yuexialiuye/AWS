# 用程式碼做出一個完整的虛擬機+網頁
程式碼網頁:https://registry.terraform.io/providers/hashicorp/aws/latest/docs/resources/instance
> 建立vpc
```sh
在test_terraform創建一個project2資料夾
在project2創建一個main.tf檔
在main.tf貼上:

provider "aws" {
  region = "us-east-1"
}

resource "aws_vpc" "myvpc"{
    cidr_block="192.168.0.0/16"
    tags = {
        Name = "myvpc"
    }
}

儲存
在cmd輸入
terraform init(初始化)
terraform apply
yes
```
